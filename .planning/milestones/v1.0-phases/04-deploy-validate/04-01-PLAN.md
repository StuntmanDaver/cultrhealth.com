# Plan 04-01: Cloudflare Pages Project Setup + DNS + Env Vars

## Objective

Set up the Cloudflare Pages project, configure DNS for staging, and set all 17 environment variables. This plan requires manual steps in the Cloudflare dashboard — it is not fully automated.

## Context

- **Source plan:** `/Users/davidk/.claude/plans/snazzy-humming-treasure.md` (Phase 5, Phase 6, Phase 8 Steps 1-2)
- **Note:** `autonomous: false` — requires human action in Cloudflare dashboard

## Tasks

### Task 1: Create GitHub repo (if not already done)

Push `cultrclub-web` to GitHub:

```bash
cd /Users/davidk/Documents/Dev-Projects/App-Ideas/cultrclub-web
# Create repo on GitHub first (via gh cli or dashboard), then:
git remote add origin https://github.com/<org>/cultrclub-web.git
git push -u origin main
git push origin staging
```

### Task 2: Create Cloudflare Pages project

In Cloudflare Pages dashboard:
1. Create project → Connect to Git → Select `cultrclub-web` repo
2. Build settings:
   - **Framework:** Next.js
   - **Build command:** `npx @cloudflare/next-on-pages@1`
   - **Build output directory:** `.vercel/output/static`
3. Set environment variable: `NODE_VERSION=20`
4. Click Deploy

### Task 3: Add DNS CNAME for staging

In **Cloudflare DNS** (for cultrhealth.com zone):
```
Type:   CNAME
Name:   staging.join
Target: staging.cultrclub.pages.dev   ← auto-assigned by Cloudflare Pages
Proxy:  DNS only (grey cloud) or Proxied
```

In **Cloudflare Pages** → cultrclub project → Custom Domains → Add domain:
- Domain: `staging.join.cultrhealth.com`
- Assign to: `staging` branch

### Task 4: Set all 17 env vars in Cloudflare Pages dashboard

**Settings → Environment Variables → Production branch:**

```
POSTGRES_URL              = <same Neon connection string as cultrhealth.com>
JWT_SECRET                = <same value as cultrhealth.com>
SESSION_SECRET            = <same value as cultrhealth.com>
RESEND_API_KEY            = <same>
FROM_EMAIL                = <same>
ADMIN_APPROVAL_EMAIL      = <same>
NEXT_PUBLIC_SITE_URL      = https://cultrclub.com
ADMIN_BASE_URL            = https://cultrhealth.com
UPSTASH_REDIS_REST_URL    = <same>
UPSTASH_REDIS_REST_TOKEN  = <same>
MAILCHIMP_API_KEY         = <same>
MAILCHIMP_AUDIENCE_ID     = <same>
MAILCHIMP_SERVER_PREFIX   = <same>
TURNSTILE_SECRET_KEY      = <same>
NEXT_PUBLIC_TURNSTILE_SITE_KEY = <same>
NODE_VERSION              = 20
```

**Settings → Environment Variables → Preview/Staging overrides:**
```
NEXT_PUBLIC_SITE_URL      = https://staging.join.cultrhealth.com
ADMIN_BASE_URL            = https://cultrhealth.com
```
