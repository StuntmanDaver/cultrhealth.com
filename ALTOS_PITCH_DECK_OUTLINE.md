## SLIDE 7: The N=1 Trial Engine

### Every Protocol = A Structured Experiment

**Protocol Engine Features:**
- Versioned protocols (e.g., "Mitochondrial Repair Protocol v1.2")
- Pre-defined expected outcomes (biomarker targets)
- Timeframe commitments (8, 12, 24 weeks)
- Adherence tracking

**Data Capture:**
```
Input (Baseline Biomarkers + Protocol + Adherence) 
  → 
Output (Outcome Biomarkers + Resilience Score Δ)
```

**Value for Altos:** Build predictive models: "If patient with phenotype X follows protocol Y, expect outcome Z"
